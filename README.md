# PrimeHunter

PrimeHunter is a backend computation engine for prime and twin-prime structural analysis.

The project is organized around three layers:

- `primehunter/math_core/` for pure mathematical helpers
- `primehunter/analysis/` for experiment and structural analysis logic
- `primehunter/data/` for exporters and report writers

Supporting project folders:

- `docs/` for architecture notes and research write-ups
- `outputs/` for generated experiment artifacts
- `tests/` for future automated coverage
- `scripts/` for future helper tooling
