# Scripts Folder

This folder holds runnable entry points and helper scripts so the repository root can stay focused on shared project files.

## Layout

- `scripts/tools/`
  Local runners for the current PrimeHunter v1 tools.

- `scripts/research/`
  Older Euclid and reciprocal-cycle experiment runners.

- `scripts/reference/`
  Reference-data generators such as the prime list writer.

## Rule

New runnable entry points should live under one of these subfolders instead of being added to the repo root.
